# Bite of Dessert — Website

Premium, fully animated marketing website for **Bite of Dessert (BOD)** — a Korean-style ice cream, lassi and pizza brand based in Aya Nagar, New Delhi.

Built with **Next.js 15 (App Router)**, **Tailwind CSS**, and **Framer Motion**.

## Quick start

```bash
# 1. Install dependencies
npm install

# 2. Run the dev server
npm run dev
# → http://localhost:3000

# 3. Production build
npm run build
npm run start
```

Tested with **Node 18.17+** (Node 20 LTS recommended).

## Folder structure

```
app/                 Next.js App Router (layout, page, globals.css, robots)
components/
  layout/            Navbar, Footer
  sections/          Hero, About, Products, Gallery, WhyChooseUs, Contact
  ui/                Loader, Reveal, SectionHeading, ProductCard, FeatureCard, GradientButton, FloatingScoop
  providers/         MotionProvider (LazyMotion + reduced-motion)
lib/
  site.ts            Brand constants — edit phone/address here, everywhere updates
  data.ts            Menu, features, gallery arrays
public/images/       Put your photos here (see table below)
```

## Replace the placeholder assets

Every image slot ships with a **branded local SVG placeholder** (no network required). They are generated by `scripts/generate-placeholders.mjs` into `public/images/{products,gallery,about,hero}/`. When real photography arrives, drop JPG/PNG files with the **same filename** and the site instantly swaps them in — no code edit needed.

| Path                                    | Replace with                                       |
|-----------------------------------------|----------------------------------------------------|
| `public/images/logo.svg`                | Provided **BOD** gold-on-black logo (keep filename `logo.svg`, or update `Navbar.tsx` for `.png`) |
| `public/images/hero/hero-scoop.svg`     | Transparent PNG of a hero scoop/sundae             |
| `public/images/products/*.svg`          | 12 product photos — keep filenames (see list below) |
| `public/images/gallery/g1.svg` … `g6.svg` | 6 gallery photos                                 |
| `public/images/about/{kitchen,plating,pizza}.svg` | 3 factory / kitchen photos                |
| `public/og.png`                         | 1200 × 630 social share image                      |

Product filenames: `korean-icecream`, `special-lassi`, `rose-lassi`, `mango-lassi`, `cold-drinks`, `corn-pizza`, `capsicum-pizza`, `onion-pizza`, `paneer-pizza`, `grilled-sandwich`, `burger-regular`, `aaloo-tikki`.

If you swap to JPG/PNG, update the `.svg` extension in `lib/data.ts` (and `components/sections/Hero.tsx` / `About.tsx` for those three).

## Sections (in order on the landing page)

1. **Hero** — headline, dual CTA, floating scoop collage, gradient orb, scroll cue
2. **About** — parallax image collage + "Hygiene First" & "Pure Ingredients" copy + stats strip
3. **Products** — Framer `layoutId` pill tabs between Drinks & Treats / Pizza Paradise / Grilled & Burger
4. **Gallery** — 6-tile asymmetric grid with hover zoom and captions
5. **Why Choose Us** — 6 feature cards with icon rotate-on-hover
6. **Contact** — form (client-only), contact card, Google Maps embed

## Animations

- `components/ui/Reveal.tsx` wraps scroll-reveal (fade + slide, stagger, `once: true`)
- `components/ui/FloatingScoop.tsx` handles infinite float/rotate on hero decorations
- Framer Motion is loaded via `LazyMotion` (`components/providers/MotionProvider.tsx`) so the `framer-motion` bundle stays small
- `prefers-reduced-motion: reduce` is respected globally (`MotionConfig reducedMotion="user"` + CSS overrides in `globals.css`)

## Contact form

The form currently resolves client-side (fake 700ms delay + success toast). To ship real submissions:

- **Formspree** — add your form ID and POST to `https://formspree.io/f/{id}`
- **Resend / SendGrid** — create an API route at `app/api/contact/route.ts`, read `FormData`, send email, return JSON
- **Google Sheet** — use a serverless function + Sheets API

Update `components/sections/Contact.tsx > onSubmit` accordingly.

## SEO

- Full `metadata` block in `app/layout.tsx` (title template, description, OpenGraph, Twitter)
- `app/robots.ts` for `/robots.txt`
- JSON-LD `Restaurant` schema injected from `app/layout.tsx` (name, address, phone, hours, cuisines)

Update `metadataBase` in `app/layout.tsx` to your production domain.

## Accessibility

- Keyboard focus rings on every interactive element (`focus-visible:ring-gold-400`)
- Semantic headings, `aria-label`s on icon-only buttons
- Images carry meaningful `alt` text
- Respects reduced-motion preference
